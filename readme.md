### qtail
qq昵称小尾巴生成器


### 如何使用
#### 示例站点
[示例站点](https://xiaozhisans.github.io/qtail)


#### 部署本地
```bash
git clone git@github.com:XiaozhiSans/qtail.git
cd qtail
```
然后运行你的本地服务器,并访问对应的页面
