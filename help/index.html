基本操作
  1. 在输入框中输入你的昵称和想要的尾巴
  2. 点击生成按钮
  3. 点击复制按钮
    4. 注意: qtail.js v1.6.2 / qtail page v0425 以后的页面版本没有复制按钮
    直接点击结果框/按下F8即可复制
  4. 进入qq修改昵称/群昵称

---------------

优雅操作
  1. tab 开始输入/下一个输入框
  2. shift + tab 返回上一个输入框
  3. enter 生成昵称
  4. f8 复制昵称
  5. esc 关闭页面

常见问答
Q:我生成的昵称有问题怎么办?
- A:更换浏览器后重试,推荐使用这些浏览器:
  • Microsoft Edge
  • Google Chrome
  • Mozilla Firefox

Q:我生成的昵称里尾巴的符号跑到昵称和尾巴中间去了?/生成的昵称尾巴符号不对?
- A:很抱歉,目前不支持结尾为中文符号/全角符号(以及unicode符号)的尾巴,你可以:
  1. 用英文符号/半角符号来代替
  2. 把结尾符号放到尾巴开头

Q:我按你说的换成了英文符号结尾符号生成出来怎么变少了
- A:很抱歉,因为生成器目前是通过split()来去除结尾符号就会导致这种情况,你可以按照上述解决方法中的第二点来解决此问题
