帮助文档 (v0520)

基本操作

  1. 在输入框中输入你的昵称和想要的尾巴
  2. 点击生成按钮
  3. 直接点击结果以复制
  4. 进入qq修改昵称/群昵称

---------------

优雅操作

  1. tab 开始输入/下一个输入框
  2. shift + tab 返回上一个输入框
  3. enter 生成昵称
  4. f8 复制昵称
  5. esc 关闭页面

---------------

常见问答

Q:我生成的昵称有问题怎么办?
- A:更换浏览器后重试,推荐使用这些浏览器:
  • Microsoft Edge
  • Google Chrome
  • Mozilla Firefox
